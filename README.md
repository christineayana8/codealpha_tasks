# CodeAlpha Data Analytics Internship Tasks

## Overview
This repository contains all 4 tasks completed for the CodeAlpha Data Analytics Internship (May Batch 2026).

## Tasks Completed

### Task 1: Web Scraping
- Scraped 400 books from books.toscrape.com
- Extracted title, price, rating, availability, genre and description
- Libraries used: requests, BeautifulSoup, pandas

### Task 2: Exploratory Data Analysis (EDA)
- Analyzed the scraped dataset for patterns and trends
- Explored price distribution, ratings, genres and outliers
- Libraries used: pandas, numpy

### Task 3: Data Visualization
- Built a 6-chart analytics dashboard
- Charts include rating distribution, price histogram, genre analysis and more
- Libraries used: matplotlib, seaborn

### Task 4: Sentiment Analysis
- Classified book descriptions as Positive, Neutral or Negative
- Analyzed sentiment trends by genre and star rating
- Libraries used: TextBlob, pandas

## Dataset
- 400 books scraped live from books.toscrape.com
- 6 columns: title, price, rating, availability, genre, description

## How to Run
1. Install dependencies: pip install requests beautifulsoup4 pandas matplotlib seaborn textblob
2. Run task1 first to generate the dataset
3. Then run tasks 2, 3 and 4 in any order
